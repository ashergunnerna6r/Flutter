# LingoChat - Flutter App Project

LingoChat is a production-grade cross-platform real-time chat application with AI live translation powered by Gemini and Firebase.

## Features
- **Firebase Authentication**: Email/Password & Google OAuth
- **Cloud Firestore**: Real-time messaging, user profiles, status tracking & emoji reactions
- **Gemini AI**: Automated real-time translation per user language preference
- **Firebase Cloud Messaging (FCM)**: Push notifications

## How to Run
1. Ensure Flutter SDK is installed: `flutter --version`
2. Run `flutter pub get`
3. Launch app: `flutter run` or build APK: `flutter build apk`
